# 更新记录

本项目采用 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 格式，并遵循语义化版本。

## [1.4.3] - 2026-06-29

### 改进

- 分享卡片改为按实际内容裁剪图片高度，内容较少的咖啡豆、年历卡片不再在底部留大片空白，并统一上下留白。
- 咖啡豆分享移除标签图片；有包装袋图片时可自行勾选是否加入分享卡片，加入则按完整比例放大显示方便识图购买，不加入则卡片不显示该区块。
- 年历分享卡片重做为按月份逐行的圆点网格，统计改为本年杯数、用豆、花费、连续天数，更贴近界面设计稿。
- 冲煮辅助页顶部计时器固定不动，仅步骤区域滚动，阶段切换时步骤列表自动定位到当前阶段并居中。
- 冲煮辅助把「当前阶段」名称与目标参数合并进计时器圆环显示，去掉下方重复的阶段卡片。

### 修复

- 冲煮辅助页去掉与关闭键重复的返回键，底部操作按钮改为右对齐并保持一行。
- 修复冲煮辅助对话框在主页底部异常显示的问题（关闭状态下不再占位）。

### 发布

- Android 版本更新为 `versionName 1.4.3`、`versionCode 26`，已生成正式签名版 `dist/coffee-vault-1.4.3-release.apk`。
- 构建前通过 `node --check www/app.js`、`node --check www/repository.js` 和 `npm.cmd test`；正式 APK 已确认未新增网络、相册或外部存储权限。

## [1.4.2] - 2026-06-29

### 新增

- 咖啡豆详情、冲煮方案详情和咖啡日历新增分享卡片，可生成统一极简票据风 PNG 并调用系统分享面板。
- 分享卡片内容与渲染样式分离，预留后续多种分享卡片样式。
- 月历分享包含月历点阵、选中日摘要和当天记录；年历分享包含年度统计和全年点状图。

### 改进

- 优化冲煮辅助底部按钮布局，三个按钮始终一行展示，并避免步骤列表被底部按钮遮挡。
- 冲煮方案详情中的“冲煮辅助”入口改为整行展示。

### 发布

- Android 版本更新为 `versionName 1.4.2`、`versionCode 25`，已生成正式签名版 `dist/coffee-vault-1.4.2-release.apk`。
- 构建前通过 `node --check www/app.js`、`node --check www/repository.js` 和 `npm.cmd test`；正式 APK 已确认未新增网络、相册或外部存储权限。

## [1.4.1] - 2026-06-28

### 改进

- 饮用详情中高级评价与评价备注前移到冲煮参数之前，且空内容不再显示占位。
- 统一弹窗底部操作按钮位置，备份弹窗按钮与标题分隔线保持间距。
- 备份导入支持 1 到当前 schema 的跨版本文件，并兼容缺失方案列表的旧备份。

## [1.4.0] - 2026-06-28

### 新增

- 新增手冲冲煮辅助，可从“喝一杯”或“冲煮方案”详情启动。
- 辅助开始前倒计时 3 秒，随后按方案分段步骤自动推进计时。

### 改进

- 喝一杯来源的辅助完成后回填本次冲煮时长，继续由用户确认评价并保存。
- 方案详情来源的辅助完成后仅展示结果，不生成饮用记录。
- 辅助页使用现有四种主题变量，避免固定深色视觉。

## [1.3.7] - 2026-06-28

### 改进

- 喝一杯的冲煮参数与方案编辑字段对齐，补齐研磨设备、目标总时长、浸泡环境、咖啡机和自定义方式。
- 饮用详情只展示当前冲煮方式相关的参数，避免手冲记录出现下压时间、浸泡时长等无关字段。
- 高级评价维度并入高级评价开关下，减少设置页割裂感。
- 咖啡日历对调“今天”和“当前选中日期”的展示形式，让今天保持更醒目的提示。

## [1.2.0] - 2026-06-24

### 新增

- 独立饮用记录系统，支持用豆克数、冲煮方式、时间、整体评分、备注与七维评价。
- “豆仓 / 饮用”双标签首页、全局饮用时间线和单款咖啡豆饮用历史。
- 卡片“喝一杯”快捷入口，可在设置中调整默认克数。
- 高级评价开关与香气、酸质、甜感、醇厚度、余韵、平衡度、苦涩感显示控制。
- 关于页，展示真实版本、产品介绍、最新功能与离线隐私说明。

### 变更

- SQLite 数据库升级至 schema v2；饮用记录的新增、修改与删除会在事务中同步调整库存。
- JSON 备份升级至 schema v2，包含豆子、饮用记录和偏好设置，并兼容 v1 备份。
- 删除咖啡豆后保留其饮用历史与豆名快照。
