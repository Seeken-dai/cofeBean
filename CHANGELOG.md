# 更新记录

本项目采用 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 格式，并遵循语义化版本。

## [1.2.5] - 2026-06-25

### 新增

- 烘焙度新增“中深烘”选项，并支持 OCR 识别“中深度烘焙 / City Roast / Medium Dark”。

### 变更

- 优化离线 OCR 解析：增强表格式标签、Origin 横排标签、风味词、净含量、产地、处理法和中文标题识别。
- 饮用页右上角搜索改为可用，可搜索豆名、冲煮方式、备注、克数和饮用时间。

## [1.2.4] - 2026-06-25

### 新增

- 设置页新增价格显示单位，可选择每克、每 50g、每 100g 或每斤单价。
- 咖啡豆编辑页可手动拍摄或读取咖啡袋、标签图片。
- OCR 新建入口新增豆仓风格图片来源面板，替代 Android 默认拍照/相册选择弹窗。

### 变更

- 图片留存继续使用 App 私有空间，取消编辑不会保存临时图片。

## [1.2.3] - 2026-06-25

### 新增

- 咖啡豆详情页自动计算并展示每克单价。
- OCR 识别后提示将当前图片保存为咖啡袋或标签。
- 咖啡豆详情页新增咖啡袋与标签图片留存区，图片保存在 App 私有空间。

### 变更

- SQLite 数据库升级至 schema v4，新增咖啡袋图片与标签图片字段。

## [1.2.2] - 2026-06-24

### 新增

- 饮用记录新增只读详情页，通过独立编辑图标进入修改状态。
- 新增豆仓主题的选项托盘，覆盖 Smart Select、烘焙度、状态和冲煮方式。
- 新增自绘月份日历与时间输入，替代 Android 原生日期时间面板。

### 变更

- 评分不再显示清空按钮；再次点击当前星级或表情等级即可取消。

## [1.2.1] - 2026-06-24

### 新增

- 新增咖啡豆只读详情页，并提供独立的编辑图标入口。
- 新增开封日期字段，旧购买日期数据继续保留以保证兼容。

### 变更

- 收紧豆仓卡片高度、间距和信息层级，提高列表空间利用率。
- “快捷喝一杯”改为带克数角标的杯形图标按钮，不再单独占用一行。

## [1.2.0] - 2026-06-24

### 新增

- 独立饮用记录系统，支持用豆克数、冲煮方式、时间、整体评分、备注与七维评价。
- “豆仓 / 饮用”双标签首页、全局饮用时间线和单款咖啡豆饮用历史。
- 卡片“喝一杯”快捷入口，可在设置中调整默认克数。
- 高级评价开关与香气、酸质、甜感、醇厚度、余韵、平衡度、苦涩感显示控制。
- 关于页，展示真实版本、产品介绍、最新功能与离线隐私说明。

### 变更

- SQLite 数据库升级至 schema v2；饮用记录的新增、修改与删除会在事务中同步调整库存。
- JSON 备份升级至 schema v2，包含豆子、饮用记录和偏好设置，并兼容 v1 备份。
- 删除咖啡豆后保留其饮用历史与豆名快照。

## [1.1.1] - 2026-06-22

### 修复

- 修正部分三星设备上拍照与从相册读取按钮文案相反的问题。
- 增强包装风味描述的离线提取效果。

## [1.1.0] - 2026-06-21

### 新增

- 使用系统相机或相册读取咖啡包装，并通过本机 ML Kit 离线识别豆名、产地、处理法、烘焙日期与风味描述。
- 识别结果置信提示与保存前人工确认。

## [1.0.1] - 2026-06-20

### 修复

- 修复部分 Android 设备首次启动时 SQLite 只读连接导致的加载失败。

## [1.0.0] - 2026-06-19

### 新增

- 咖啡豆 CRUD、搜索、筛选、排序、统计与 Smart Select 批量管理。
- 完全离线的 SQLite 存储、JSON 备份恢复、四套外观主题和旧版 localStorage 数据迁移。
- Android 状态栏、安全区域、启动画面与系统返回键适配。
