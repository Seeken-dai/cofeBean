豆仓 Coffee Vault

一包豆子 · 一段风味 · 一页日常

为咖啡爱好者设计的本地优先记录工具。保存每包豆子的产地、风味与价格，记录每次冲煮与饮用，让日常里喝过的咖啡慢慢有迹可循。

功能

从一包豆子到一杯咖啡，都记下来

☕

咖啡豆管理

产地、处理法、烘焙度、每克单价与赏味期倒计时；按状态筛选、多维度排序，赏味将至与余量不足自动提醒。

📷

拍照识别（离线）

APK 内置 ML Kit 模型，离线识别中英文包装标签，自动解析日期、克重、处理法、烘焙度与风味，低可信字段高亮待确认。

📅

饮用记录与统计

「喝一杯」自动扣减余量；多维风味雷达图，咖啡日历提供月历详情与年度格子热力图，当天花费与评分一目了然。

⏱️

冲煮方案与辅助

按冲煮方式管理方案，饮用参数可另存为新方案；冲煮辅助按步骤自动计时，突出本段注水量并常驻下一段预览。

🎫

分享

咖啡豆、冲煮方案、咖啡日历生成统一票据风分享卡片；方案还能生成离线分享码（DC1-）与二维码，扫码即可导入。

☁️

云同步（可选）

默认关闭，不登录不联网。登录后在 Android 与 Web 间同步豆子、记录、方案与图片，合并采用 last-write-wins，可随时删除云端。

离线与隐私

你的咖啡数据，先属于你

默认完全离线：数据保存在你的设备（Android SQLite / Web IndexedDB），不需要账号或网络，拍照识别只申请相机权限。云同步是可选功能，仅在登录并开启后使用；数据经 HTTPS 传输、仅你自己可见，你可以随时退出账号或删除云端副本。
